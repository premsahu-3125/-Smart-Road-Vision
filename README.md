# # 🚦 Smart Road Vision – Real-Time Road Object Detection



Smart Road Vision is an AI-powered desktop application designed to detect and classify road objects in real time using computer vision and deep learning. The application processes images, videos, webcam feeds, and image folders through an intuitive desktop interface, making road scene analysis efficient and accessible.





## 📌 Overview

The project combines YOLOv5, OpenCV, and PyTorch with a modern Tkinter desktop interface to identify multiple road objects such as vehicles, pedestrians, traffic lights, and other common road elements.
Designed with usability in mind, Smart Road Vision includes built-in analytics, configurable detection settings, activity logging, screenshot capture, and report generation.

## 🚀 Getting Started

Follow these steps to set up and run the project locally.

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/premsahu-3125/SmartRoadVision.git
cd SmartRoadVision
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Launch the Application

```bash
python main.py
```
